# Changelog

## 1.0.0

Initial release of GBA Emulator for VS Code.

### Features

- Embedded Game Boy Advance emulator inside the VS Code Activity Bar
- Dedicated webview panel for gameplay
- ROM loading from local files in `.gba` and `.bin` formats
- Pause and resume controls
- Reset and stop controls
- In-emulator screenshot capture
- Internal save state support with session persistence
- Save state export and import via `.gbastate`
- Standard `.sav` export and import
- Optional automatic `.sav` file output in the workspace
- Adjustable emulator volume
- Keyboard-based controller mapping for GBA inputs
- Built-in minimal BIOS support
- Multilingual UI support
- Debug logging for troubleshooting emulator behavior
